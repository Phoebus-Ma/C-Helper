### Introduction

UDP multicast test sample code.


### Platform

Ubuntu for Linux.


### Build

```console
make
```


### Run

`Note: .`

```console
$ ./udp-server

UDP server running.
[Server] client ip: 192.168.6.85, port: 39921.
[Server] Recv data: Hello UDP Server.
[Server] Recv data: Hello UDP Server.
```

```console
$ ./udp-client

[Client] Recv data: OK.
```
